munshibaba.com has three (03) level accounting system. At the main dashboard, four (04) basic accounting heads have been provided for the convenience of the users. User can create unlimited number of main/basic heads even at the level.

It is better to make a chart of accounts first on paper before creating sub-categories / sub-heads and then create further accounts in those sub heads. For instance, in Bank, user can create different banks, and then in those banks user can create multiple accounts.
